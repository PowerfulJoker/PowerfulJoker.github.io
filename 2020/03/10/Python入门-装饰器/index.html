Python

在代码运行期间动态增加功能的方式，称之为“装饰器”（Decorator）。本质上，decorator就是一个返回函数的高阶函数。

注意，在学习装饰器之前，你需要知道对象、函数嵌套(在函数中定义另一个函数)和高阶函数的概念。

一切皆对象

由于函数也是一个对象，而且函数对象可以被赋值给变量，所以，通过变量也能调用该函数。
